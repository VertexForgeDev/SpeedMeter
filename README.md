# ⚡ SpeedMeter | Connection Diagnostics

An ultra-fast, modern Progressive Web App (PWA) designed for high-precision internet speed testing. Built with a responsive glassmorphism interface, it delivers real-time connection telemetry through a lightweight, standalone single-page setup.

## ✨ Features

* **Instant Diagnostic Integration:** Employs precise real-time speed metrics tracking down download, upload, and latency components.
* **Premium Glassmorphic Design:** Optimized with dynamic CSS radial gradients, subtle noise filtration overlay textures, and fluid animations for an immersive desktop and mobile layout.
* **Progressive Web App (PWA) Capable:** Includes comprehensive support for PWA criteria, featuring cross-platform app shell capability, cache-busting configurations (`manifest.json?v=1.2`), and native Service Worker initialization.
* **Native Platform Share API:** Quick diagnostics sharing utilizing browser-level fallback compatibility checks.
* **Optimized Mobile Viewport:** Fully responsive architecture that adapts gracefully to structural display variables across touch devices (`viewport-fit=cover`).

## 🛠️ Tech Stack

* **Frontend:** HTML5, modern vanilla JavaScript (ES6+), semantic architecture
* **Styling:** Custom CSS Custom Variables, Google Fonts (`Plus Jakarta Sans`)
* **Tracking & Optimization:** Native Service Worker APIs, embedded Web Analytics Integration
